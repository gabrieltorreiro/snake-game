# Jogo Snake

> Essa é uma implementação de um jogo popular conhecido como snake (jogo da cobrinha).

**O jogo da cobrinha (snake) foi criado em 1976 e após isso surgiram diversas versões.** <br>
**É um jogo simples, porem divertido.**
